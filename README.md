# LumiKids

Frontend React/Vite preparado para deploy na Vercel consumindo um backend separado.

## Variáveis de ambiente

```env
VITE_API_BASE_URL=https://lumikids-backend.vercel.app
```

## Scripts

```bash
npm run dev
npm run test
npm run build
```

## Deploy na Vercel

- importe este repositório sozinho na Vercel
- configure `VITE_API_BASE_URL` apontando para o backend publicado
